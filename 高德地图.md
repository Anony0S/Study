## 报错

`Uncaught Error: Invalid Object: LngLat(NaN, NaN)`

![image-20230201104013627](C:\Users\Admin\Documents\Typora\高德地图.assets\image-20230201104013627.png)

解决：

```less
.index {
  width: 100%;
  height: 100vh; // 高度使用vh，不能使用100%
  position: relative;
  #container {
    width: 100%;
    height: 100%;
    padding: 0px;
    margin: 0px;
    position: absolute;
  }
}
```
